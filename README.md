# eRO-ExTra catalog
A catalog of 304 **Ex**tragalactic non-AGN X-ray **Tra**nsients and variables detected with **eRO**SITA telescope during its first and second all-sky surveys. This repository provies data, key scripts and visualisations for the eRO-ExTra catalog.

The full catalog description and methodology is presented in the publication:

📚🔗 [Grotova et al., 2025, Astronomy & Astrophysics](https://www.aanda.org/articles/aa/pdf/2025/01/aa51253-24.pdf)
